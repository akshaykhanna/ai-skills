# AI Skills Repository

This repository contains custom skills for Antigravity agents to toggle between different execution modes:

- **[read-only](read-only/SKILL.md)**: Configures the agent to run in a safe, read-only mode where modifications to code and the environment are prohibited.
- **[action](action/SKILL.md)**: Configures the agent to run in action/write mode, allowing files to be edited and system modifying commands to be executed.
